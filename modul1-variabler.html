Modul 1 – Variabler og datatyper

Læringsmål

Etter denne leksjonen skal du kunne:

  • Opprette variabler og gi dem verdier
  • Gjenkjenne de fire vanligste datatypene: str, int, float og bool
  • Bruke type() til å sjekke hva slags verdi du har
  • Følge Python sine navneregler for variabler

Hva er en variabel?

Tenk på en variabel som en merket eske. Du putter noe oppi esken, setter en lapp på den med et navn — og når du vil ha tak i det igjen, ber du bare om esken med det navnet.

navn = "Ola"

Her lager vi en variabel som heter navn og putter teksten "Ola" oppi. Likhetstegnet (=) betyr ikke «er lik» — det betyr «lagre dette».

Når du så skriver navn et annet sted i koden, henter Python frem verdien «Ola».

De fire viktigste datatypene

Python har mange typer verdier. De fire du trenger å kjenne til nå:

1. Tekst — str (string)

Tekst skrives alltid inni anførselstegn. «str» er kort for «string» (streng).

fornavn = "Ola" etternavn= "Nordmann" hilsen = "God morgen!"

2. Heltall — int (integer)

Heltall skrives uten anførselstegn og uten desimaler.

alder = 15 antall = 42 temperatur= -3 # negative tall fungerer også

3. Desimaltall — float

Desimaltall skrives med punktum (ikke komma — Python er internasjonalt).

pi = 3.14159 pris = 49.90 karakter = 5.5

4. Sannhetsverdi — bool (boolean)

En bool kan bare være to ting: True (sant) eller False (usant). Legg merke til stor forbokstav!

er_student = True har_bil = False er_voksen = False
Aldri skriv true eller false med liten bokstav. Python vil ikke forstå det og gi deg en feil.

Prøv alle fire typene

Sjekke typen med type()

Lurer du på hvilken type en variabel er? Bruk type():

Du skal se:

<class 'str'> <class 'int'> <class 'float'> <class 'bool'>

Navneregler for variabler

Python er streng på hva variabler kan hete:

Bruk kun bokstaver, tall og understrek (_). Start alltid med en bokstav eller understrek. Bruk beskrivende navn: alder_til_bruker er bedre enn a.
Ikke bruk mellomrom (mitt navn er ugyldig). Ikke start med et tall (1variabel er ugyldig). Ikke bruk norske tegn (æ, ø, å) i variabelnavn — det kan skape problemer.
✅ Gyldig ❌ Ugyldig
fornavnfor navn (mellomrom)
alder_til_bruker1alder (starter med tall)
_privatmy-variable (bindestrek)
maks_poenghøyde (norsk tegn)

Variabler kan endres

En variabel er ikke låst — du kan gi den en ny verdi når som helst:

Mange variabler på én linje (valgfritt)

Python lar deg sette mange variabler på én linje, adskilt med komma. Dette er ikke nødvendig, men du vil kanskje se det i andres kode:

x, y, z = 1, 2, 3

🪪 Oppgave

Lag et program i VSCode (idkort.py) som lagrer opplysninger om deg og printer dem pent.

Programmet skal printe noe som ligner:

  • Bruk én variabel per opplysning
  • Alle fire datatypene (str, int, float, bool) skal brukes minst én gang
