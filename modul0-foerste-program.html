Modul 0 – Ditt første Python-program

Læringsmål

Etter denne leksjonen skal du:

  • Vite hva print() gjør og hvordan du bruker den
  • Forstå strukturen til en enkel Python-fil
  • Ha laget og kjørt ditt eget Python-program i VSCode
  • Prøvd Python direkte i nettleseren med den innebygde terminalen

Hva er et program?

Et Python-program er rett og slett en tekstfil som slutter på .py. Filen inneholder instruksjoner — én per linje. Python leser dem fra topp til bunn og utfører dem i rekkefølge.

La oss starte med den enkleste instruksjonen: print().

print() — skriv tekst til skjermen

print() er en funksjon — noe Python vet fra før at den kan gjøre. Du gir den en verdi inni parentesene, og den viser verdien i terminalen.

print("Hei verden!")
Hei verden!

Du kan printe så mange linjer du vil:

print("Linje 1") print("Linje 2") print("Linje 3")
Linje 1 Linje 2 Linje 3

Tekst må alltid stå inni anførselstegn — enten enkle ' eller doble ":

print('Enkle anførselstegn fungerer også!')
Python bryr seg ikke om du bruker enkle eller doble anførselstegn — velg én stil og hold deg til den. I dette kurset bruker vi doble.

Prøv det selv — innebygd terminal

Endre koden nedenfor og klikk ▶ Kjør kode for å se resultatet umiddelbart. Du trenger ikke åpne VSCode for dette — terminalen her er direkte i nettleseren.

Prøv å endre teksten inni anførselstegnene og kjør koden på nytt. Hva skjer om du glemmer anførselstegnene rundt teksten?

Lag programmet i VSCode

Nettleseterminalen er fin for å prøve, men ekte Python-programmer lager du i VSCode. Slik gjør du det:

  1. Åpne VSCode og mappen din (python-kurs).
  2. Klikk Ny fil i filpanelet og kall den hei.py.
  3. Skriv dette:
    print("Hei!") print("Jeg heter ...") print("I dag lærer jeg Python.")
  4. Åpne terminalen (Ctrl + `) og kjør:
    python hei.py
  5. Du skal se de tre linjene i terminalen. 🎉

Hva skjer om det går galt?

Python er nøyaktig. Det vanligste nybegynnerfeil er:

print(Hei verden) # FEIL — tekst uten anførselstegn
SyntaxError: invalid syntax
Print("Hei") # FEIL — stor forbokstav
NameError: name 'Print' is not defined
Python skiller mellom store og små bokstaver. print og Print er to helt forskjellige ting. Bruk alltid små bokstaver.
