Modul 0 – Python og VSCode på Windows

Læringsmål

Etter denne leksjonen skal du ha:

  • Installert Python 3 på Windows-PCen din
  • Installert Visual Studio Code
  • Lagt til Python-utvidelsen i VSCode
  • Bekreftet at alt fungerer ved å sjekke versjonen

Del 1 – Installer Python

Python er selve programmet som forstår og kjører koden din. Uten det skjer ingenting.

  1. Åpne nettleseren og gå til python.org/downloads. Klikk på den store gule knappen — den velger riktig versjon for Windows automatisk.
  2. Kjør installasjonen. Det første bildet du ser er det viktigste:

    Huk av for «Add Python to PATH» helt nederst på det første bildet før du klikker Installer. Uten dette finner ikke terminalen Python.
    Klikk deretter Install Now.
  3. Vent til installasjonen er ferdig. Klikk Close.

Sjekk at Python fungerer

Åpne ledetekst (søk etter «cmd» i Start-menyen) og skriv:

python --version

Du skal se noe som ligner:

Python 3.12.4
Nøyaktig versjonsnummer (3.12, 3.11, osv.) spiller ingen rolle — bare at det starter med 3. Ser du noe som «Python 2.x», har du en gammel versjon. Last ned ny fra python.org.

Del 2 – Installer Visual Studio Code

VSCode er editoren du skriver kode i. Den er gratis, rask og veldig populær blant utviklere.

  1. Gå til code.visualstudio.com og klikk Download for Windows.
  2. Kjør filen du lastet ned. Godta lisensavtalen og klikk deg gjennom installasjonen. Du kan la alle innstillingene stå som de er.
  3. La «Add to PATH» være avhuket (det er det vanligvis). Klikk Install og deretter Finish. VSCode åpnes automatisk.

Del 3 – Legg til Python-utvidelsen

VSCode støtter hundrevis av programmeringsspråk, men du må installere støtte for hvert språk separat.

  1. I VSCode, klikk på ikonet som ser ut som fire firkanter på venstre side (eller trykk Ctrl + Shift + X). Dette åpner Extensions.
  2. Søk etter Python. Den første treffet fra Microsoft (blå verifisert merke) er den riktige. Klikk Install.
  3. Etter installasjon: trykk Ctrl + Shift + P og søk etter «Python: Select Interpreter». Velg Python 3.x fra listen. Da vet VSCode hvilken Python-versjon den skal bruke.
Installer gjerne også «Pylance» — det gir deg bedre autofullføring og hjelp mens du skriver kode.

Del 4 – Sjekk at alt henger sammen

La oss bekrefte at VSCode og Python kommuniserer:

  1. Opprett en ny mappe på skrivebordet og kall den python-prosjekter.
  2. I VSCode: Fil → Åpne mappe og velg den nye mappen.
  3. Opprett en ny fil: klikk pluss-ikonet i filpanelet til venstre. Kall den test.py.
  4. Skriv dette i filen:
    print("Python fungerer! 🎉")
  5. Åpne terminalen inne i VSCode: Terminal → Ny terminal (eller Ctrl + `).
  6. I terminalen, skriv:
    python test.py
    Du skal se:
    Python fungerer! 🎉
Ser du en feil? Det vanligste problemet er at «Add Python to PATH» ikke ble huket av under installasjonen. Avinstaller Python og installer på nytt — husk å huke av boksen!
På noen Windows-maskiner heter kommandoen python3 istedenfor python. Prøv begge om den ene ikke fungerer.
