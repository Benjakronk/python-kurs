Modul 2 – If, elif og else

Læringsmål

  • Skrive if-setninger som tar beslutninger basert på betingelser
  • Bruke elif og else for flere valgmuligheter
  • Bruke sammenligningsoperatorer (==, !=, <, >, <=, >=)
  • Kombinere betingelser med and, or og not

If — ta en beslutning

En if-setning kjører kodeblokken inni bare om betingelsen er sann. Legg merke til kolon og innrykk:

Sammenligningsoperatorer

OperatorBetyrEksempel
==er likx == 5
!=er ikke likx != 0
<mindre ennx < 10
>større ennx > 0
<=mindre/likx <= 18
>=større/likx >= 0
== sjekker om to ting er like. = lagrer en verdi. De er veldig forskjellige!
